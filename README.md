# ROS - Initial

### Initial ROS project setup

The following should be added to your bash.rc or zsh equivalent
```
source <Absolute Path>/HyperRail/devel/setup.bash  

export PYTHONPATH="<Absolute Path>/HyperRail/src/hyper_rail/src"
```
